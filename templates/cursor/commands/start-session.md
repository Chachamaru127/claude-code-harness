# /start-session - セッション開始

Cursor (PM) としてのセッションを開始し、現在の状況を把握します。

---

## 使い方

```
/start-session
```

または自然言語で：
- 「今日の作業を始めたい」
- 「状況を確認して」
- 「何から始めればいい？」

---

## 実行フロー

### Step 1: 状態確認

以下のファイルを確認して現在の状況を把握:

```bash
# Plans.md から現在のタスク状況を確認
cat Plans.md

# 最近の git 履歴を確認
git log --oneline -5

# 未コミットの変更を確認
git status -sb
```

### Step 2: 状況サマリーの作成

```markdown
## 📊 現在の状況

**日時**: {{YYYY-MM-DD HH:MM}}
**ブランチ**: {{current_branch}}

### タスク状況
- 🔴 進行中: {{wip_count}}件
- 🟡 未着手: {{todo_count}}件
- 🟢 完了待ち確認: {{done_count}}件

### 最近の活動
{{recent_commits}}

### 未コミットの変更
{{uncommitted_changes}}
```

### Step 3: 次のアクション提案

状況に応じて提案:

| 状況 | 提案 |
|------|------|
| 進行中タスクあり | 「Claude Code で `/start-task` を実行して続きを」 |
| 完了待ちタスクあり | 「`/review-cc-work` でレビューしましょう」 |
| 未着手タスクのみ | 「`/assign-to-cc` でタスクを依頼しましょう」 |
| タスクなし | 「`/plan-with-cc` で計画を立てましょう」 |

---

## 出力例

```
📊 **セッション開始** - 2025-01-15 09:00

**現在の状況:**
- 🔴 進行中: 1件「ユーザー認証機能の実装」
- 🟡 未着手: 3件
- 🟢 確認待ち: 0件

**最近の活動:**
- abc1234 feat: ログイン画面の実装 (2時間前)
- def5678 fix: バリデーションエラー修正 (昨日)

**提案:**
進行中のタスクがあります。
→ Claude Code で「続きをお願い」と言うか `/start-task` を実行

または、新しいタスクを依頼する場合は `/assign-to-cc` を使用してください。
```

---

## 関連コマンド

| コマンド | 用途 |
|---------|------|
| `/plan-with-cc` | 計画を立てる・相談する |
| `/assign-to-cc` | タスクを依頼する |
| `/review-cc-work` | 完了タスクをレビュー |
| `/project-overview` | プロジェクト全体の状況 |
