## 第1条（定義）

本契約において、以下の用語は以下の意味を有するものとします。

1. **本ソフトウェア**: Claude Code Harness、並びにこれに含まれるすべてのソースコード、ドキュメント、設定ファイル、スクリプト、テンプレート、その他の構成要素を指します。
2. **利用者**: 本ソフトウェアを取得し、本契約に同意した個人または法人を指します。
3. **再配布**: 本ソフトウェアの全部または一部を、有償・無償を問わず、第三者に対して譲渡、販売、配布、公開、または利用可能な状態に置く行為を指します。
4. **類似サービス**: 本ソフトウェアと同一または類似の機能・目的を有する開発ツール、プラグイン、テンプレート、フレームワーク、またはサービスを指します。

## 第2条（利用許諾）

著作権者は、以下の条件の下で、利用者に対して本ソフトウェアの利用を許諾します。

### 2.1 許可される利用

利用者は、以下の行為を無償で行うことができます。

- **個人利用**: 個人的なプロジェクトでの使用
- **商用利用**: 商業目的での使用（受託開発、自社プロダクト開発、フリーランス業務など）
- **改変**: 本ソフトウェアを編集、カスタマイズ、拡張すること
- **統合**: 本ソフトウェアを他のソフトウェアやシステムと組み合わせて使用すること
- **クライアントワーク**: 第三者（クライアント）のために本ソフトウェアを用いた開発を行うこと
- **AI（LLM）による編集・生成**: 生成AI（Large Language Model）を用いて本ソフトウェアを編集、拡張、またはコード生成の補助として利用すること（詳細は第4条を参照）

## 第3条（禁止事項）

利用者は、以下の行為を行ってはなりません。

### 3.1 再配布・販売の禁止

- 本ソフトウェアの全部または一部を、有償・無償を問わず、**再配布、販売、譲渡、公開、または第三者が利用可能な状態に置くこと**を禁止します。
- これには、以下の行為が含まれますが、これらに限定されません:
  - 本ソフトウェアをそのまま、または軽微な変更を加えて販売・配布すること
  - 本ソフトウェアをテンプレートとして配布すること
  - 本ソフトウェアをオープンソースリポジトリ、マーケットプレイス、プラグインストア、その他の配布プラットフォームに公開すること
  - 本ソフトウェアを含むパッケージ、ツール、フレームワークを販売・配布すること

### 3.2 類似サービスの提供禁止

- 本ソフトウェアを基盤として、または参考にして、**類似サービス、競合プラグイン、テンプレート販売サービス、またはこれに準ずるサービス**を提供することを禁止します。
- これには、以下の行為が含まれますが、これらに限定されません:
  - 本ソフトウェアと類似の開発ハーネス、プラグイン、ワークフロー管理ツールを作成・販売すること
  - 本ソフトウェアのコード、設計、アーキテクチャを模倣した製品・サービスを提供すること

### 3.3 著作権の主張禁止

- 本ソフトウェアの著作権を利用者が主張すること、または著作権者の表示を削除・改変することを禁止します。

### 3.4 公序良俗に反する利用の禁止

- 本ソフトウェアを、以下のような公序良俗に反する目的で使用することを禁止します:
  - 違法行為、詐欺、犯罪行為の支援
  - アダルトコンテンツ、ポルノグラフィ、わいせつ物の作成・配信
  - ヘイトスピーチ、差別、誹謗中傷、ハラスメント
  - マルウェア、ウイルス、スパイウェア、その他の悪意のあるソフトウェアの作成・配布
  - その他、法令または公序良俗に反する行為

### 4.2 利用者の責任

- AI（LLM）を使用して生成または編集されたコードについて、**利用者は自己の責任において検証、テスト、デバッグを行う**ものとします。
- AIサービスの利用規約、データセキュリティ、プライバシーポリシーの遵守は、**利用者の責任**とします。
- AI生成コードに起因する不具合、セキュリティ脆弱性、第三者との紛争について、著作権者は一切の責任を負いません。
