# タスク: review-security

## 概要
コードのセキュリティレビューを実施する。

## テスト対象機能
- `/harness-review` コマンド
- review スキル（並列サブエージェント）
- セキュリティ問題の検出

## 前提条件
`src/api/auth-handler.ts` が存在すること（意図的な脆弱性を含む）

## プロンプト

```
src/api/auth-handler.ts のセキュリティレビューを実施してください。

重点確認項目:
- SQL インジェクション
- XSS（クロスサイトスクリプティング）
- 認証・認可の問題
- 機密情報の露出

問題を発見した場合は、修正案も提示してください。
```

## 期待される出力
- セキュリティ問題のリスト
- 各問題の重大度（Critical/High/Medium/Low）
- 修正案の提示

## 成功基準
| 基準 | 条件 |
|------|------|
| 問題検出 | 3個以上の問題を検出 |
| 重大度分類 | 各問題に重大度がある |
| 修正案 | Critical/High には修正案がある |
| 具体性 | 問題箇所（行番号）が特定されている |

## 測定ポイント
- レビュー完了までの時間
- 検出された問題数
- サブエージェント起動有無（並列レビュー）
