# タスク: review-quality

## 概要
コードの品質レビューを実施する。

## テスト対象機能
- `/harness-review` コマンド
- review スキル（並列サブエージェント）
- コード品質問題の検出

## 前提条件
`src/components/Dashboard.tsx` が存在すること（品質問題を含む）

## プロンプト

```
src/components/Dashboard.tsx の品質レビューを実施してください。

重点確認項目:
- コードの可読性
- パフォーマンス（不要な再レンダリング）
- ベストプラクティス違反
- 保守性の問題

改善提案も含めてください。
```

## 期待される出力
- 品質問題のリスト
- 各問題の影響度
- 改善提案

## 成功基準
| 基準 | 条件 |
|------|------|
| 問題検出 | 3個以上の問題を検出 |
| カテゴリ分類 | 問題がカテゴリ分けされている |
| 改善提案 | 具体的な改善コードがある |
| 実用性 | 提案が実際に適用可能 |

## 測定ポイント
- レビュー完了までの時間
- 検出された問題数
- 改善提案の具体性スコア
