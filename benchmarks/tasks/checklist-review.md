# チェックリスト付きレビュータスク

## 目的
完了度を測定可能なベンチマーク用タスク

## プロンプト

```
以下のチェックリストに従って、3つのファイルをレビューしてください。
各項目について「完了」または「未完了」を明記してください。

## 必須タスク（全12項目）

### auth-handler.ts セキュリティレビュー
- [ ] 1. SQLインジェクションの問題を特定し、行番号を記載
- [ ] 2. SQLインジェクションの修正コードを提示
- [ ] 3. パスワード関連の問題を特定し、行番号を記載
- [ ] 4. パスワード関連の修正コードを提示

### Dashboard.tsx 品質レビュー
- [ ] 5. useEffect関連の問題を特定し、行番号を記載
- [ ] 6. useEffect関連の修正コードを提示
- [ ] 7. 型定義の問題を特定し、行番号を記載
- [ ] 8. 型定義の修正コードを提示

### user-service.js リファクタリング提案
- [ ] 9. モジュール形式の問題を特定
- [ ] 10. ES Modules形式への変換例を提示
- [ ] 11. 非同期処理の問題を特定
- [ ] 12. async/awaitへの変換例を提示

## 回答フォーマット
各項目について以下の形式で回答してください：

[x] 1. SQLインジェクション - 行12: `SELECT * FROM...`
   修正: パラメータ化クエリを使用

最後に完了数をサマリーしてください：
完了: X/12
```

## 評価方法
- 完了数（0-12）をカウント
- 実行時間を測定
- 効率スコア = 完了数 / 実行時間（秒） × 100
