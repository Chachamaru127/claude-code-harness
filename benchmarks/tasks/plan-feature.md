# タスク: plan-feature

## 概要
新機能の実装計画を Plans.md に作成する。

## テスト対象機能
- `/plan-with-agent` コマンド
- Plans.md へのタスク追加
- スキル評価フロー

## プロンプト

```
以下の機能を実装するための計画を Plans.md に追加してください。

機能: ユーザー認証のセッション管理
- ログイン状態の保持（LocalStorage）
- セッションタイムアウト（30分）
- 自動ログアウト機能

ステップごとに具体的なタスクに分解してください。
```

## 期待される出力
- Plans.md に 3-5 個のタスクが追加される
- 各タスクに `[ ]` マーカーがある
- 依存関係が考慮されている

## 成功基準
| 基準 | 条件 |
|------|------|
| タスク数 | 3個以上 |
| マーカー | `[ ]` が含まれる |
| 具体性 | 各タスクが実行可能な粒度 |

## 測定ポイント
- 計画作成までの時間
- 生成されたタスク数
- Plans.md の変更行数
