# Status Summary

進捗状況のサマリーを人間（特に VibeCoder）が読みやすい形式で生成するスキル。

---

## 目的

集計結果と異常検知結果を組み合わせて：
- わかりやすい進捗レポートを生成
- VibeCoder 向けの平易な言葉で説明
- 次のアクションを明確に提示

---

## 入力

| 項目 | 説明 |
|------|------|
| `total_tasks` | 総タスク数 |
| `progress_percent` | 進捗率 |
| `current_work` | 現在作業中のタスク |
| `next_task` | 次に着手すべきタスク |
| `warnings` | 検知された警告 |

---

## 出力

| 項目 | 説明 |
|------|------|
| `user_message` | ユーザー向けサマリーメッセージ |

---

## 出力フォーマット

```markdown
## 📊 進捗サマリー

**更新日時**: {{timestamp}}
**ブランチ**: {{current_branch}}

---

### タスク状況

| ステータス | 件数 |
|-----------|------|
| 🔴 未着手 | {{tasks_todo}} |
| 🟡 作業中 | {{tasks_wip}} |
| 🟢 完了 | {{tasks_done}} |
| 🚫 ブロック | {{tasks_blocked}} |

**進捗率**: {{progress_percent}}%

---

### 現在の作業中タスク
{{current_work}}

### 次に着手すべきタスク
{{next_task}}

{{warnings}}
```

---

## 進捗の視覚表現

### プログレスバー

```
進捗: [████████░░░░░░░░░░░░] 40%
```

### ステータス絵文字

| 進捗率 | 表示 |
|--------|------|
| 0-25% | 🚀 スタートしたばかり |
| 26-50% | 💪 順調に進行中 |
| 51-75% | 🎯 ゴールが見えてきた |
| 76-99% | ✨ もうすぐ完了！ |
| 100% | 🎉 全タスク完了！ |

---

## VibeCoder 向け説明

### 技術用語の言い換え

| 技術用語 | VibeCoder 向け |
|---------|---------------|
| WIP | 作業中 |
| blocked | 止まっている（理由あり） |
| dependency | 他の作業待ち |
| merge | 変更を統合 |
| deploy | 公開する |

### 次のアクション提示

```markdown
### 💡 次にできること

- 「続けて」→ 作業中のタスクを続行
- 「次のタスク」→ 次のタスクを開始
- 「レビューして」→ 完了した作業をチェック
- 「計画を見せて」→ 全体の計画を確認
```

---

## 警告の統合

警告がある場合は末尾に追加：

```markdown
---

### ⚠️ 注意事項

1. 2 件のタスクが同時に進行中です
   - 1 つに集中すると効率が上がります

2. 「外部API連携」が 72 時間止まっています
   - 問題があれば教えてください
```

---

## 出力例

```markdown
## 📊 進捗サマリー

**更新日時**: 2024-01-15 14:30
**ブランチ**: staging

---

### タスク状況

進捗: [████████████░░░░░░░░] 58%

| ステータス | 件数 |
|-----------|------|
| 🔴 未着手 | 3 |
| 🟡 作業中 | 1 |
| 🟢 完了 | 7 |
| 🚫 ブロック | 1 |

💪 順調に進行中です！

---

### 🔧 現在の作業中
**ユーザー認証機能の実装**
- 開始: 今日 10:30
- 見込み: 今日中に完了予定

### 📋 次のタスク
**検索機能の追加**
- 優先度: 高（Cursor から依頼済み）

---

### 💡 次にできること

- 「続けて」→ 認証機能の続きを作業
- 「状況を教えて」→ 詳細な進捗を確認

---

### ⚠️ 注意事項

「外部API連携」が 3 日間止まっています。
解決が必要であれば教えてください。
```

---

## 注意事項

- 専門用語を避け、平易な日本語で説明
- 絵文字を適度に使って視認性を向上
- 次のアクションを必ず提示
